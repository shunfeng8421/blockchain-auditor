# Blockchain Auditor ⛓️🔐

**6-stage automated audit pipeline for blockchain and cryptocurrency repositories.**

## Pipeline
```
Discover → Audit → Contribute → Track → Case Study → Daily Report
 (weekly)   (label)   (label)     (every 4h)  (label)      (daily)
```

## 20 Audit Rules
Categories: Solidity/EVM security, DeFi vulnerabilities, supply chain risks, tokenomics.

## Quick Start
```bash
python discover.py --limit 10
python audit.py --dir /path/to/contracts/
```
